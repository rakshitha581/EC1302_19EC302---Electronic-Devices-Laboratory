**[AIM]{.underline}**

To learn Resistor Colour Code and to determine the stated value of a
resistor by interpreting the colour code indicated on the resistor.

**[APPARATUS REQUIRED]{.underline}**

Resistors -- 5 No's.

**[THEORY]{.underline}**

The colour bands on the body of the resistor tell how much resistance it
has. As shown in the following diagrams figure (1), there is a 4-band
resistors. The 4-band resistors, the last band indicates tolerance in
table (1). Consult with the "Resistor colour codes and Resistor
Tolerance" in table (2) chart for finding the tolerance value.
